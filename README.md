# 🐍 SnakeGame

This application was developed as part of an academic project for the User Interface (UI) subject. 


It's a modernized version of a classic **Snake game** implememted as a console application written in C# (.NET Framework) featuring a keyboard-controlled menu and sound effects, and an optional AI‑powered configuration mode.

Players can set up the game manually or describe their desired settings in natural language — the app then interprets the description using the Gemini API (Flash‑Lite model) and generates the configuration automatically.

## Features

- **Keyboard‑controlled application**. No input required
- **AI‑assisted configuration**. Describe the game you want in plain language, and the AI generates the settings
- **Manual customization**. Choose number of fruits, size and look of the board, snake color and speed
- **Sound effects**. Different sounds for intro, option selection, eating and game ending
- **Randomized food spawning**  
- **Score tracking**    

## Tech stack

- **Language:** C#
- **Framework:** .NET 4.8.1
- **AI Integration**: Gemini API (gemini-2.5-flash-lite)
- **Libraries:** Figgle (Generates ASCII art text banners), Pastel (Adds color to console text), Spectre.Console (Building rich console interfaces)
- **IDE:** Visual Studio
